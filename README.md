## Travel Planner

Travel Planner je spletna aplikacija za načrtovanje potovanj, ki uporabnikom omogoča organizacijo destinacij, aktivnosti, proračuna in časovnic na enem mestu.

## Funkcionalnosti

Dodajanje in upravljanje destinacij
Načrtovanje poti po dnevih (itinerary)
Spremljanje stroškov in proračuna
Seznam stvari za pakiranje
Prikaz vremena za izbrane lokacije
Uporabniški računi (prijava / registracija)
Tehnologije

## Projekt uporablja:

Frontend: HTML, CSS, JavaScript (React)
Backend: Node.js
Baza podatkov: Sanity

## Kriterji za projekt

- Admin panel
  - Pregled stavri
  - Dodajanje
  - Filtreranje
  - Sortiranje

- Vizaulni podatki
  - Graf
  - Preglednica

- Uvoz podatkov v bazo preko .csv ali .excel
- Komunikacija z strankami
- Avtomatski pošiljanje mail-ov
- Izvoz PDF datoteke
- Uporaba zunanjih API-ov

## Installation

1. Clone the repository
2. Run `npm install`
3. Start the server with `npm start`
